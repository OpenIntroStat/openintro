# Function to extract OpenIntro IMS colors as hex codes

Uses full colors from
[IMSCOL](https://openintrostat.github.io/openintro/reference/IMSCOL.md)

## Usage

``` r
openintro_cols(...)
```

## Arguments

- ...:

  Character names of
  [openintro_colors](https://openintrostat.github.io/openintro/reference/openintro_colors.md)

## Examples

``` r

openintro_cols("blue")
#>      blue 
#> "#569BBD" 
openintro_cols("red")
#>       red 
#> "#F05133" 
```
