# Turkey tourism

Summary of tourism in Turkey.

## Usage

``` r
tourism
```

## Format

A data frame with 47 observations on the following 3 variables.

- year:

  a numeric vector

- visitor_count_tho:

  a numeric vector

- tourist_spending:

  a numeric vector

## Source

Association of Turkish Travel Agencies, Foreign Visitors Figure &
Tourist Spendings By Years.
<https://www.tursab.org.tr/en/statistics/foreign-visitors-figure-tourist-spendings-by-years_1083.html>
