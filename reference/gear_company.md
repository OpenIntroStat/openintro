# Fake data for a gear company example

Made-up data for whether a sample of two gear companies' parts pass
inspection.

## Usage

``` r
gear_company
```

## Format

A data frame with 2000 observations on the following 2 variables.

- outcome:

  a factor with levels `not` `pass`

## Examples

gear_company
