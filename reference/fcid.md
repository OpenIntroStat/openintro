# Summary of male heights from USDA Food Commodity Intake Database

Sample of heights based on the weighted sample in the survey.

## Usage

``` r
fcid
```

## Format

A data frame with 100 observations on the following 2 variables.

- num_of_adults:

  a numeric vector
