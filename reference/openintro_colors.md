# OpenIntro colors

A character string of full colors from `IMSCOL[,1]`

## Usage

``` r
openintro_colors
```

## Format

A named character string with 9 elements: "blue", "green", "pink",
"yellow", "red", "black", "gray", "lgray

## Examples

``` r

openintro_colors
#>      blue     green      pink    yellow       red     black      gray     lgray 
#> "#569BBD" "#114B5F" "#D89A9E" "#F4DC00" "#F05133" "#000000" "#808080" "#D9D9D9" 

openintro_colors["blue"]
#>      blue 
#> "#569BBD" 
```
