# Return function to interpolate an OpenIntro IMS color palette

Not exported

## Usage

``` r
openintro_pal(palette = "main", reverse = FALSE, ...)
```

## Arguments

- palette:

  Character name of palette in
  [openintro_palettes](https://openintrostat.github.io/openintro/reference/openintro_palettes.md)

- reverse:

  Boolean indicating whether the palette should be reversed

- ...:

  Additional arguments to pass to
  [`grDevices::colorRampPalette()`](https://rdrr.io/r/grDevices/colorRamp.html)
